# 颜色警报功能 - 快速启动指南

## 30 秒快速开始

1. **打开 OnTopReplica 应用**
2. **右键点击主窗口** → 选择 **"Color Alert"**
3. **点击 "Choose..."** 选择你想监控的颜色（例如红色）
4. **勾选** "Enable Color Detection"
5. **关闭面板** - 现在应用开始监控！

当窗口中出现你选择的颜色时，会听到 **3 秒的报警声**。

## 详细步骤

### 第一步：启动颜色警报

```
主窗口（显示你要监控的窗口克隆）
    ↓
右键点击
    ↓
选择菜单中的 "Color Alert"
    ↓
颜色警报面板打开
```

### 第二步：选择颜色

```
颜色警报面板
    ↓
找到 "Target Color" 部分
    ↓
点击 "Choose..." 按钮
    ↓
在颜色对话框中选择颜色
    ↓
点击 OK 确认
```

示例：
- **游戏**：选择敌人或道具的颜色
- **数据监控**：选择状态指示灯的颜色
- **网页**：选择提示信息的背景色

### 第三步：调整设置

#### 颜色容差(Color Tolerance)

```
含义：允许颜色的变化范围
范围：0 (精确匹配) → 255 (匹配任何颜色)

┌─────┬───────────────┬─────┐
│ 0   │ ━━━━━●━━━━━   │ 255 │ 当前: 30
└─────┴───────────────┴─────┘
      低                  高
    精确           宽松匹配
```

推荐值：
- **精确颜色识别**：10-20
- **一般应用（推荐）**：25-40
- **宽松匹配**：50-100

#### 采样间隔(Sample Interval)

```
含义：多久检查一次窗口（毫秒）

输入框显示：[500]
范围：100-10000ms

┌────────┬────────┬──────────┐
│ 100ms  │ 500ms  │ 1000ms   │
│ 很灵敏 │ 平衡   │ 省资源   │
└────────┴────────┴──────────┘
推荐使用 300-800ms
```

推荐值：
- **响应要求高**：300-500ms
- **一般应用（推荐）**：500-1000ms
- **后台监控**：1000-2000ms

### 第四步：启用检测

```
勾选复选框
☑ Enable Color Detection

颜色警报现已激活！
```

### 第五步：关闭面板

```
点击 "Close" 按钮
    ↓
返回主窗口
    ↓
开始监控！
```

## 常见操作

### 切换颜色

1. 打开颜色警报面板
2. 点击 "Choose..." 选择新颜色
3. 关闭面板保存

### 调整敏感度

```
太灵敏（误报多）
    ↓
增加容差值 (例如 30 → 50)
    ↓
再次测试

灵敏度不够
    ↓
减小容差值 (例如 50 → 30)
```

### 暂时禁用

```
☑ Enable Color Detection
    ↓
点击取消勾选
☐ Enable Color Detection
    ↓
监控停止（但设置保留）
```

### 快速重新启用

```
打开面板（Ctrl+右键）
勾选启用复选框
关闭面板
```

## 故障排除

### "听不到报警声"

✓ **检查项**：
1. 系统音量是否开启？
2. 应用是否被静音？
3. 颜色是否真的出现在窗口中？

✓ **解决方案**：
- 调整 Windows 系统音量
- 检查应用程序音量混合器设置
- 减小容差值以更准确地匹配颜色
- 增加 CPU 监控 log 检查是否检测到颜色

### "频繁误报"

✓ **原因**：
容差值设置过高

✓ **解决方案**：
减小容差值（例如 50 → 30）

### "反应太慢"

✓ **原因**：
采样间隔太长

✓ **解决方案**：
减小采样间隔（例如 1000ms → 500ms）

### "CPU 占用太高"

✓ **原因**：
采样间隔太短

✓ **解决方案**：
增加采样间隔（例如 300ms → 1000ms）

## 高级技巧

### 技巧 1：使用 HTML 颜色代码

```
选择颜色后，记住十六进制值 (例如 #FF5733)
下次可以快速重现相同配置
```

### 技巧 2：针对不同场景保存配置

```
思路：
1. 为场景 A 配置颜色警报
2. 记录颜色代码、容差、间隔
3. 需要场景 A 时快速重新输入

✓ 避免每次重新调参
```

### 技巧 3：组合使用与其他功能

```
颜色警报 + 区域选择
→ 只监控窗口的特定部分

颜色警报 + 占比调整
→ 监控不同大小的颜色区域

颜色警报 + 全屏模式
→ 在全屏下接收警报
```
